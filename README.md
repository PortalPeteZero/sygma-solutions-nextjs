# Sygma Solutions Next.js Migration

Next.js 14 App Router migration of sygma-solutions.com with infrastructure, shared components, and SEO configuration.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Project Structure

- `src/app/` - App Router pages and layouts
- `src/components/` - Shared React components
- `src/lib/` - Utilities and helpers
- `src/data/` - Static data (knowledge hub items, etc.)
- `public/` - Static assets

## Configuration

- Next.js 14 with App Router
- Tailwind CSS with Radix UI components
- TypeScript strict mode
- 465+ redirects from legacy site
